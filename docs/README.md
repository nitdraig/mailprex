# Mailprex Documentation

Nextra documentation site for Mailprex.

## Development

```bash
npm install
npm run dev
```

Content lives in [`content/`](content/). Navigation is configured in `content/_meta.json`.

## Build

```bash
npm run build
npm run start
```

Deployed at [docs.mailprex.excelso.xyz](https://docs.mailprex.excelso.xyz).
