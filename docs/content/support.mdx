## Common issues

**401 Invalid form token**  
Use the full `mk_live_…` token. Regenerate in the dashboard if you only saved the prefix.

**403 Destination email must match**  
`emailDestiny` in the request must equal your Mailprex account email.

**429 Rate limit / quota**  
Free plan: 200 sends/month. Upgrade via dashboard when Gumroad is enabled.

**Legacy UUID token**  
Dashboard shows a migration banner — click **Regenerate secure token** and update every site using the old value.
